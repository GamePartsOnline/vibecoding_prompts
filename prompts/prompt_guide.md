# Guide des Prompts Efficaces

## La structure d'un prompt (RCCF)
* **Rôle** : L'identité ou l'expertise que l'IA doit adopter.
* **Contexte** : Les informations de fond nécessaires pour cadrer la tâche.
* **Consigne** : L'instruction précise de ce que l'IA doit faire.
* **Format** : La structure souhaitée pour le résultat (tableau, code, liste, etc.).

## Techniques avancées
* **Zero-shot** : Demander sans exemple. Utile pour des questions directes.
* **Few-shot** : Fournir 1 à 3 exemples pour guider le modèle vers le style ou le format voulu.
* **Chain-of-Thought (CoT)** : Inciter l'IA à "réfléchir étape par étape" pour résoudre des problèmes complexes.
