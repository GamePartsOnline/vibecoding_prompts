# vibecoding_prompts
## Prompt Library
The `prompts/` folder contains templates designed to speed up application development:
- `prompt_guide.md`: A reminder of prompt engineering best practices.
- `html_prompt.txt`: HTML starter template.
- `rails_prompt.txt`: Rails API initialization commands.
- `story_prompt.txt`: User story generation.

*How to use them: Copy the content of the desired file, adapt any bracketed placeholders if needed, and paste it into your LLM.*
